# AburunGo — Learning Paths Plan

Research-grounded plan to turn AburunGo's existing buckets (words, characters, phrases, flashcards,
audio, Hana) into retention-optimized **learning paths**, without replacing those buckets.

**Spine:** JLPT level ladders (N5 → N1), each running one shared retention engine; emphasis shifts
from recognition (N5) to production/immersion (N3+). Free through N4, paid from N3.
